# Non-iterative-optimal-blind-deconvolution
This is a Matlab code library for non iterative blind deconvolution methods.It solves the problem of traditional blind deconvolution with non convex optimization difficulty and inability to obtain stable global optimal solutions. It is the first to use positive samples in blind deconvolution algorithms.
Ref: He L , He J , Zhou Q ,et al.Non-iterative optimal blind deconvolution and its application to machine condition monitoring[J].Mechanical Systems and Signal Processing, 2026, 243(c):113715.DOI:10.1016/j.ymssp.2025.113715.
